# 🚦 RoadSafe Analytics – Road Accident Analysis

## 📌 1. Title

**Road Accident Data Analysis using Power BI**

---

## ❗ 2. Problem Statement

Road accidents are a major public safety concern. Understanding patterns, causes, and high-risk factors is essential to improve road safety.
This project aims to analyze road accident data and provide meaningful insights using interactive dashboards.

---

## 📂 3. Dataset Description

The dataset consists of three main tables:

* **Accidents Data:** Contains details about accident events
* **Casualties Data:** Information about injured or deceased individuals
* **Vehicles Data:** Details of vehicles involved in accidents

### Data Structure:

* Raw data is stored in the `Data/Raw_Data` folder
* Cleaned data is stored in the `Data/Processed_Data` folder

---

## 🧹 4. Data Cleaning & Transformation

Data preprocessing was performed using **Power BI (Power Query)**:

* Removed null values
* Removed duplicate records
* Renamed columns for better readability
* Fixed data types (date, numeric, categorical)
* Removed unnecessary columns

---

## 🔗 5. Data Modeling

* Established relationships between datasets using **Accident_Index**
* Created a structured data model for analysis

---

## 📊 6. KPI Matrix

Key performance indicators used in the dashboard:

* 🚗 Total Accidents
* ⚠️ Total Casualties
* 💀 Fatal Accidents
* 📈 Accident Severity Distribution
* 🚙 Accidents by Vehicle Type
* 🗺️ Accidents by Location
* 🕒 Time-based Trends

---

## 📈 7. Dashboard Description

The Power BI dashboard provides:

* Overview of accident statistics
* Region-wise and category-wise analysis
* Time trend analysis (monthly/yearly)
* Severity and vehicle analysis
* Interactive filters for dynamic exploration
