# Week 1 - Data Understanding

## Objective

The objective of this week was to understand the structure and contents of the road accident dataset.

## Dataset Overview

The dataset consists of three main tables:

* **Accidents Data:** Contains information about accident events such as date, time, severity, and location
* **Casualties Data:** Contains details about individuals affected in accidents
* **Vehicles Data:** Contains information about vehicles involved in accidents

## Key Columns Identified

* Accident_Index (Unique identifier for each accident)
* Number_of_Vehicles
* Number_of_Casualties
* Accident_Severity
* Date and Time
* Vehicle_Type
* Casualty_Class

## Activities Performed

* Loaded all datasets into Power BI
* Reviewed column names and data types
* Identified relationships between tables
* Understood the meaning of important fields

## Observations

* Data contains missing values in several columns
* Some columns use coded values instead of descriptive labels
* Multiple tables are interconnected using Accident_Index

## Outcome

A clear understanding of the dataset structure and relationships was achieved, which will help in further data processing and analysis.
