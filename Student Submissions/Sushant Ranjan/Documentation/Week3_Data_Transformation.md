# Week 3 - Data Transformation

## Objective

The objective of this week was to transform the cleaned data into a suitable format for analysis and visualization.

## Activities Performed

* Converted columns into appropriate data types (date, time, numeric)
* Created new columns such as Month and Year from date
* Filtered and organized relevant data for analysis
* Standardized values across different columns
* Prepared datasets for building relationships and dashboard usage

## Tools Used

* Power BI (Power Query Editor)

## Challenges Faced

* Handling inconsistent data formats
* Ensuring correct transformation without data loss
* Creating meaningful derived columns

## Outcome

The dataset was successfully transformed and structured, making it ready for data modeling and analysis.
