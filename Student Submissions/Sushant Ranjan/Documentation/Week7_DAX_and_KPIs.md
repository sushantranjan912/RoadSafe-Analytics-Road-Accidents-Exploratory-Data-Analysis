# Week 7 - DAX and KPI Development

## Objective

The objective of this week was to create measures and key performance indicators (KPIs) using DAX for analysis.

## Activities Performed

* Created DAX measures for key metrics
* Calculated total accidents and casualties
* Developed KPIs for accident severity analysis
* Used DAX functions to perform calculations and aggregations

## Key Measures Created

* Total Accidents
* Total Casualties
* Fatal Accidents
* Accident Severity Count

## Tools Used

* Power BI (DAX)

## Challenges Faced

* Understanding DAX syntax and functions
* Creating accurate calculations
* Debugging incorrect results

## Outcome

Important KPIs were created using DAX, which helped in building meaningful and interactive dashboard visuals.
